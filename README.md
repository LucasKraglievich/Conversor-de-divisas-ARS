# Conversor de Divisas ARS 💱

Conversor de monedas con cotizaciones en tiempo real del dólar Oficial, Blue y USDT Cripto para Argentina.

## ✨ Features
- Cotizaciones en tiempo real vía dolarapi.com
- Conversión entre ARS, USD, BRL y EUR
- Tres tipos de cambio: Oficial, Blue y USDT Cripto

## 🛠 Stack
HTML · CSS · JavaScript · Fetch API

## 📌 Cómo usarlo
Cloná el repo, abrí index.html con Live Server y listo.
